3 Data visualisation

This list provide resources for those who want to produce fancy figure and tables when communicating their results. Data visualisation is an important aspect of statistics and requires careful thought about how to appropriately and effectively present your results in a visually pleasing manner.

3.1 Journal articles and textbooks

Wilkinson, L. (2005). The grammar of graphics (2nd Edition). Springer.

  • Textbook that outlines foundation of graphing and is utilised in the R ggplot2 package. Available online: https://sydney.primo.exlibrisgroup.com/permalink/61USYD_INST/12rahnq/alma991017882639705106

Wickham, H. (2016). Ggplot2: elegant graphics for data analysis (2nd Edition). Springer.

  • Guide and manual for the ggplot2 package for R. Very useful graphing package. Available online: https://sydney.primo.exlibrisgroup.com/permalink/61USYD_INST/1c0ug48/alma991014142669705106

Winston, C. (2018). R graphics cookbook: practical recipes for visualising data (2nd Edition). O’Reilly Media.

  • Provide useful examples on how to visualise data and produce useful graphs. Primarily uses the ggplot2 package. Available online: https://sydney.primo.exlibrisgroup.com/permalink/61USYD_INST/2rsddf/cdi_askewsholts_vlebooks_9781491978573

Healy, K. (2019). Data visualization: A practical introduction. Princeton University Press.

  • Introductory textbook on graphing and data viz. Focuses on ggplot2 and the tidyverse.

Healy, K., & Moody, J. (2014). Data visualisation in sociology. Annu. Rev. Sociol. 40: 150-128.

  • Article outlining data viz specific to sociology. Good place to see how to work towards a consistent standard for graphing sociological insights.

Ola, O., & Sedig, K. (2016). Beyond simple charts: design of visualisations for big health data. Online J Public Health Inform 8(3): e195.

  • Provides a framework for designing graphs associated with “big” data.

Slaughter, S. J., & Delwiche, L. D. (2010). Using PROC SGPLOT for quick high-quality graphs. SAS Global Forum.

  • SUGI paper on graphing with SAS… if that’s your thing. https://support.sas.com/resources/papers/proceedings10/154-2010.pdf

Mitchell, M. N. (2012). A visual guide to stata graphics. Stata Press.

  • Introductory textbook for graphing using STATA. Also has accompanying website https://www.stata.com/support/faqs/graphics/gph/stata-graphs/

3.2 Websites

https://www.jscarlton.net/post/2015-10-24visualizinglogistic/

  • Demonstrates one way to present odds ratios and CIs graphically.

https://www.r-graph-gallery.com/

  • Provides lots of inspiration and example code for graphing in R.

https://www.tableau.com/

  • Online platform for data graphing and visualization.

https://dreamrs.github.io/esquisse/index.html

  • Package that allows you to interactively design graphs quickly in R.

https://www.data-to-viz.com/

  • Awesome webpage that provides a nice decision tree around what graph to use for the type of data you have.

https://seaborn.pydata.org/

  • This is a large data visualisation library for Python.

https://rstudio.github.io/leaflet/ or https://leafletjs.com/

  • Open-source JavaScript library for interactive maps.

https://bookdown.org/ndphillips/YaRrr/pirateplot.html

  • Example of a pirate plot that provides more usable information on a single graph than the traditional box plot.

https://stats.idre.ucla.edu/spss/seminars/spssgraphics/graphics-in-spss/

  • Graphing in SPSS… if that’s your thing.

https://stats.idre.ucla.edu/sas/modules/graphing-data-in-sas/

  • More info on graphing in SAS.

https://stats.idre.ucla.edu/stata/modules/graph8/intro/introduction-to-graphs-in-stata/

  • Graphing in Stata… if that’s your thing.

https://data.library.virginia.edu/getting-started-with-shiny/

  • Website that provides and introduction to Shiny (a package that lets you create interactive web apps to visualise data).

https://shiny.rstudio.com/images/shiny-cheatsheet.pdf

  • Shiny cheat sheet

https://www.rstudio.com/wp-content/uploads/2015/03/ggplot2-cheatsheet.pdf

  • Data viz cheat sheet in ggplot2

https://plotly.com/

  • Similar to shiny and lets you create web apps to vizualise data. Can use Python, R codes.

https://plotly.com/graphing-libraries/

  • Extensive graph library in Python, R, and Javascript. Provides lots of inspiration.

https://www.gapminder.org/tools/#$chart-type=bubbles

  • Fun online tool to explore how powerful data viz can be.

https://developers.google.com/chart

  • Google charts. This is more for developing interactive charts for websites/mobile devices.
